bScript - The lightweight js framework

Aims

  • Be lightweight
  • Make javascript easy
  • Replace jQuery on small projects
  • Support ie7 + (there are a few bugs in ie7 at present)

Css

Usage

$(selector).css(propertyName,Value);

This method can be chained.

Example

$('.example-css').css('color','red').css('textDecoration','underline');
